ℹ️ About RAGita

RAGita is an advanced AI-powered question-answering system for the Bhagavad Gita, built using Retrieval-Augmented Generation (RAG) technology.

Key Features:

  • Zero Hallucinations: 100% grounded answers with verifiable citations
  • Multiple Personas: Choose from 7 different response styles
  • Precise Citations: Every answer backed by specific verse references
  • Semantic Search: Advanced retrieval using state-of-the-art embeddings
  • Real-time Verification: All claims verified against source text

Technology Stack:

  • FastAPI backend with RAG pipeline
  • Sentence Transformers for embeddings
  • FAISS for vector search
  • Cross-encoder reranking
  • LLM-powered generation with verification
